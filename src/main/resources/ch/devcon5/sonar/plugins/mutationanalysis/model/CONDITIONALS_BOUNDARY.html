Conditionals Boundary Mutator (CONDITIONALS_BOUNDARY)

This operator replaces relational operators <, <=, >, >= with their conditional boundary counterparts according to the following table

Original Mutation
< <=
<= <
> >=
>= >

For example

if (a <= b) {
  // some code
}

will be mutated to

if (a < b) {
  // someCode
}

see Pitest Mutators
